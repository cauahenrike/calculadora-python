# 🧮 Calculadora em Python

Este projeto é uma calculadora simples desenvolvida em Python, que permite realizar operações matemáticas básicas e armazenar o histórico das contas realizadas.

---

## 🚀 Funcionalidades

* ➕ Soma
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão (com tratamento de erro)
* 📜 Histórico de cálculos
* 🔁 Menu interativo no terminal

---

## 🛠️ Tecnologias utilizadas

* Python
* Estruturas condicionais (`if/elif/else`)
* Loops (`while`)
* Listas (armazenamento de histórico)

---

## 🧠 Como funciona

O sistema:

1. Exibe um menu interativo
2. Permite escolher entre calcular ou ver o histórico
3. Solicita dois números e um operador
4. Realiza o cálculo
5. Armazena o resultado no histórico
6. Permite visualizar todas as operações realizadas

---

## ▶️ Como executar

1. Execute o arquivo:

```bash
python calculadora.py
```

2. Escolha uma opção no menu:

```
[1] - Calcular
[2] - Histórico
[3] - Sair
```

---

## 📌 Exemplo de uso

```
Seu Primeiro numero? 10
Digite o operador (+, -, *, /): *
Seu Segundo numero? 5

Resultado: 50
```

### Histórico:

```
10 * 5 = 50
8 + 2 = 10
```

---

## ⚠️ Tratamento de erros

* ❌ Divisão por zero não é permitida
* ⚠️ Operadores inválidos são tratados
* 🔁 Retorno ao menu em caso de erro
